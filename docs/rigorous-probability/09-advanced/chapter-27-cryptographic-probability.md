# 第27章 暗号理論のための確率論

## この章で学ぶこと

暗号理論では、多くの議論が有限集合上の一様分布と確率的アルゴリズムで進む。そのため、毎回ルベーグ測度や一般の可測空間を前面に出す必要はない。しかし、ランダム変数、分布、独立性、統計距離、coupling、negligible function は、測度論的確率論の言葉で正確に整理できる。

## 前提知識

- 有限確率空間
- 確率変数と分布
- 独立性
- 全変動距離
- セキュリティパラメータに関する漸近記法

## 27.1 有限確率空間と一様分布

暗号理論で最もよく現れる確率空間は、有限集合上の一様分布である。たとえば $\{0,1\}^n$ 上の一様分布 $U_n$ は

$$
P(U_n=x)=2^{-n}
$$

で定まる。

:::tip[直感]
暗号理論の乱数は、まず有限長ビット列としてモデル化される。そのため、多くの基礎的議論では有限確率空間で十分である。
:::

### 例：ワンタイムパッド

平文 $m\in\{0,1\}^n$、鍵 $K\sim U_n$ に対して暗号文を

$$
C=m\oplus K
$$

で定める。このとき固定した $m$ に対して $C$ は一様分布に従う。完全秘匿性は、暗号文の分布が平文に依存しないこととして表される。

## 27.2 ランダム変数としてのアルゴリズムの出力

確率的アルゴリズム $A$ は、入力 $x$ と内部乱数 $R$ を使って出力を返す写像として見られる。

$$
A(x;R)
$$

$R$ を一様分布に従う確率変数と見れば、$A(x;R)$ も確率変数である。

:::info[定義：アルゴリズムの出力分布]
固定した入力 $x$ に対し、確率的アルゴリズム $A$ の出力分布とは、内部乱数 $R$ の分布を $A(x;\cdot)$ で押し出した確率測度である。
:::

これは第12章の「分布は押し出し測度である」という見方の直接の応用である。

## 27.3 確率的アルゴリズム

確率的アルゴリズムの確率は、入力のランダム性ではなく、内部乱数に関する確率として定義されることが多い。鍵生成、暗号化、署名、攻撃者の選択はいずれも確率的アルゴリズムとして表される。

### 例：暗号化の分布

公開鍵暗号の暗号化を

$$
\mathrm{Enc}(pk,m;R)
$$

と書く。固定した公開鍵 $pk$ と平文 $m$ に対して、暗号文は $R$ によってランダムに変わる。したがって暗号文空間上の分布が得られる。

## 27.4 negligible function

暗号理論の安全性は、セキュリティパラメータ $\lambda$ を大きくしたときの攻撃成功確率で定義される。

:::info[定義：negligible function]
関数 $\nu:\mathbb{N}\to[0,\infty)$ が negligible であるとは、任意の多項式 $p$ に対して、十分大きい $\lambda$ で

$$
\nu(\lambda)<\frac1{p(\lambda)}
$$

が成り立つことをいう。
:::

negligible probability とは、セキュリティパラメータの関数として negligible な確率である。固定された「小さい確率」ではない。

### 例

$2^{-\lambda}$ は negligible である。一方、$1/\lambda$ や $1/\lambda^{10}$ は negligible ではない。後者は多項式の逆数そのものだからである。

## 27.5 statistical distance

有限集合 $S$ 上の2つの分布 $P,Q$ に対して、statistical distance は

$$
\Delta(P,Q)=\frac12\sum_{x\in S}|P(x)-Q(x)|
$$

で定義される。これは全変動距離である。

### 例：有限分布での計算

$S=\{0,1\}$、$P(0)=1/2$、$P(1)=1/2$、$Q(0)=3/4$、$Q(1)=1/4$ とする。このとき

$$
\Delta(P,Q)=\frac12\left(\frac14+\frac14\right)=\frac14.
$$

statistical distance が negligible であるとは、分布族 $P_\lambda,Q_\lambda$ について $\Delta(P_\lambda,Q_\lambda)$ が negligible function になることをいう。

## 27.6 coupling

:::info[定義：coupling]
確率測度 $P,Q$ の coupling とは、周辺分布がそれぞれ $P,Q$ になるような同時分布である。すなわち、確率変数 $(X,Y)$ が $X\sim P$、$Y\sim Q$ を満たすとき、$(X,Y)$ の分布を $P,Q$ の coupling という。
:::

coupling は、2つの分布を同じ確率空間上で比較する方法である。

:::tip[直感]
分布 $P$ と $Q$ が近いことを示したいとき、同じ乱数から $X\sim P$ と $Y\sim Q$ を作り、$X\ne Y$ となる確率を小さく抑える、という考え方が coupling である。
:::

有限集合では、適切な coupling を使うと

$$
\Delta(P,Q)\le P(X\ne Y)
$$

を得られる。最良の coupling では等号も達成できる。

## 27.7 hybrid argument

hybrid argument は、2つの分布や実験を一度に比較せず、中間列を挟んで少しずつ比較する方法である。

$$
H_0 \to H_1 \to H_2 \to \cdots \to H_k
$$

$H_0$ が実世界、$H_k$ が理想世界を表すとする。各隣接ペア $H_i,H_{i+1}$ の差が小さければ、三角不等式により全体の差も抑えられる。

$$
\Delta(H_0,H_k)\le \sum_{i=0}^{k-1}\Delta(H_i,H_{i+1})
$$

:::warning[注意]
hybrid の数 $k$ がセキュリティパラメータに対して多項式個なら、各差が negligible であることから和も negligible になる。しかし指数個の hybrid を足す場合は、この結論は自動ではない。
:::

## 27.8 random oracle modelの確率論的見方

random oracle model では、ハッシュ関数を完全にランダムな関数として理想化する。有限の入力集合 $D$ と出力集合 $R$ を考えるなら、ランダムオラクルは関数集合 $R^D$ 上の一様分布から選ばれたランダム関数である。

攻撃者がクエリした入力に対しては、同じ入力には同じ出力を返し、未問い合わせの入力には新しい一様乱数を割り当てる、という形で実装的に考えられる。

:::warning[注意]
random oracle model は理想化モデルである。現実のハッシュ関数が本当にランダム関数であるという意味ではない。
:::

## 27.9 測度論が必要な場面・不要な場面

有限ビット列、有限回の実験、有限時間の攻撃者だけを扱うなら、多くの場合は有限確率空間で十分である。実際、暗号理論の基本定義は有限分布と多項式時間アルゴリズムを中心に書ける。

一方で、次の場面では測度論的な整理が役立つ。

1. 無限列や無限回の相互作用を理想化する場合
2. random oracle を関数空間上の確率測度として扱う場合
3. 分布族の収束や距離を統一的に扱う場合
4. 情報理論的安全性を KL ダイバージェンスや相互情報量で述べる場合

## 確率論での意味

暗号理論の安全性定義は、実験の出力分布を比較する形で書かれる。したがって、確率変数、押し出し測度、全変動距離、独立性、条件付き確率は、暗号理論の基礎語彙である。

## よくある誤解

- negligible probability は「かなり小さい定数」ではない。
- 暗号理論で測度論が常に前面に出るわけではない。
- statistical distance と KL ダイバージェンスは同じ量ではない。
- random oracle model は現実のハッシュ関数そのものの数学的証明ではなく、理想化された安全性モデルである。

## この章の要点

- 暗号理論の多くは有限確率空間上で定式化できる。
- 確率的アルゴリズムの出力は、内部乱数の押し出し分布である。
- negligible function は任意の多項式の逆数より速く小さくなる関数である。
- statistical distance は有限分布上の全変動距離である。
- coupling と hybrid argument は、分布の近さを証明するための基本技法である。
- 測度論は、無限対象や情報理論的量を扱うときに特に効く。

## 演習問題

1. ★ $\{0,1\}^3$ 上の一様分布で、特定のビット列が出る確率を求めよ。
2. ★ $\nu(\lambda)=2^{-\lambda}$、$\mu(\lambda)=1/\lambda^5$ が negligible function かどうか判定せよ。
3. ★★ $P=(1/2,1/2)$、$Q=(3/4,1/4)$ の statistical distance を計算せよ。
4. ★★ ワンタイムパッドで、固定した平文 $m$ に対して $C=m\oplus K$ が一様分布になることを示せ。
5. ★★ 確率的アルゴリズム $A(x;R)$ の出力分布が押し出し測度であることを説明せよ。
6. ★★★ coupling を使って、2つの分布が近いことを示す考え方を自分の言葉で説明せよ。
7. ★★★ hybrid argument で hybrid の数が多項式個であることがなぜ重要か、negligible function の和に触れて説明せよ。
8. ★★★ random oracle model が理想化モデルである理由を、現実のハッシュ関数との違いから説明せよ。
