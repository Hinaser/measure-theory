# 厳密な確率論のための測度論入門

この教材は、素朴な確率論から一歩進んで、確率論を測度論の言葉で厳密に理解することを目標にする。

同時に、測度論がどのような問題意識から生まれたのかも重視する。Fourier解析、三角級数、Cantor の点集合論、Borel 集合、Lebesgue 積分は、数学内部の困難に答える中で発展した。その道具が、連続分布、無限試行、測定誤差、統計モデル、情報や暗号の確率的議論を扱うための言葉にもなる。

中心となる問いは次である。

- 測度論はどのような歴史的・数学的問題から生まれたのか。
- 事象とは何か。
- 確率とは何を測っているのか。
- 確率変数はなぜ可測関数なのか。
- 期待値はなぜルベーグ積分なのか。
- 独立性、条件付き期待値、収束は測度論でどう定式化されるのか。
